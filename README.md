# AI-: нейросеть уровня 0 (супер базовая)

Это **реальная** нейронная сеть, но максимально простая:
- 2 входа
- 2 нейрона в скрытом слое
- 1 выход
- обучается вручную через backpropagation (без TensorFlow/PyTorch)

Сеть учится на задаче XOR.

## Что нужно
- Python 3.8+

## Как запустить
```bash
python3 ai0.py
```

Ожидаемый результат: предсказания близки к [0, 1, 1, 0].

## Как протестировать
```bash
python3 -m unittest -v
```

Тест проверяет, что после обучения сеть правильно классифицирует все 4 точки XOR.

## Как это работает очень коротко
1. `forward`: считаем выходы скрытого и выходного слоя через сигмоиду.
2. Считаем ошибку (MSE).
3. Через производные сигмоиды считаем градиенты.
4. Обновляем веса градиентным спуском.
